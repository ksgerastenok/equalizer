# Equalizer
This repository contains:
- Equalizer (20-bands) WinAmp plugin written in Free Pascal (Delphi dialect)
- Equalizer (10-bands) Quintessential Player plugin written in Free Pascal (Delphi dialect)
- Enhancer (aka Bass/Treble Enhancer + Stereo Enhancer + Dynamic Volume Normalizer) WinAmp plugin written in Free Pascal (Delphi dialect)
- Enhancer (aka Bass/Treble Enhancer + Stereo Enhancer + Dynamic Volume Normalizer) Quintessential Player plugin written in Free Pascal (Delphi dialect)
- Dynamic Volume Normalizer WinAmp plugin written in Free Pascal (Delphi dialect)
- Dynamic Volume Normalizer Quintessential Player plugin written in Free Pascal (Delphi dialect)
- Equalizer (10-bands) QMP plugin written in C++ (C20 standard)
- Enhancer Quintessential Player plugin written in C++ (C20 standard)
- Dynamic Volume Normalizer Quintessential Player plugin written in C++ (C20 standard)

The Enhancer plugin has similar functional as Enhancer v0.17 by Iosif Adryan.
The Equalizer and Enhancer are based on BiQuad IIR Filter (Peaking EQ and Bass/Treble Shelf).
More information can be found in files Audio-EQ-Cookbook.txt and Audio-EQ-Zoelzer.txt.
