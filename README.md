# Equalizer
This repository contains:
- Equalizer (10-bands + Stereo Expander + Virtual Bass + Dynamic Volume Normalizer) Quintessential Player plugin written in Free Pascal (Delphi dialect)
- Enhancer (Bass/Treble Enhancer + Stereo Expander + Virtual Bass + Dynamic Volume Normalizer) Quintessential Player plugin written in Free Pascal (Delphi dialect)
- Equalizer (20-bands + Stereo Expander + Virtual Bass + Dynamic Volume Normalizer) WinAmp / AIMP plugin written in Free Pascal (Delphi dialect)
- Enhancer (Bass/Treble Enhancer + Stereo Expander + Virtual Bass + Dynamic Volume Normalizer) WinAmp / AIMP plugin written in Free Pascal (Delphi dialect)
- Equalizer (10-bands + Stereo Expander + Virtual Bass + Dynamic Volume Normalizer) Quintessential Player plugin written in C++ (C20 standard)
- Enhancer (Bass/Treble Enhancer + Stereo Expander + Virtual Bass + Dynamic Volume Normalizer) Quintessential Player plugin written in C++ (C20 standard)

The Enhancer plugin has similar functional as Enhancer v0.17 by Iosif Adryan.
The Equalizer and Enhancer are based on BiQuad IIR Filter (Peaking EQ and Bass/Treble Shelf).
More information can be found in files Audio-Equalizer-RBJ.txt / Audio-Equalizer-Zoelzer.txt and Audio-VirtualBass.txt.
